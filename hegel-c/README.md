# hegeltest-c

C-ABI bindings for [hegel](https://hegel.dev)'s native property-based
testing engine. The build artefact is a shared library (`libhegel.so`,
`libhegel.dylib`, `hegel.dll`) plus a checked-in C header at
`include/hegel.h` (generated by `cbindgen` from `src/lib.rs`).

**This crate is published mostly to reserve the name on crates.io.**
If you want to *use* libhegel from a non-Rust language, you almost
certainly want the prebuilt cdylib + header from a GitHub release
rather than building this crate yourself:

> <https://github.com/hegeldev/hegel-rust/releases>

Each release publishes `libhegel-<goos>-<goarch>.<ext>` and a matching
`.sha256` sidecar for `linux/amd64`, `linux/arm64`, `darwin/amd64`,
`darwin/arm64`, `windows/amd64`, and `windows/arm64`.
